# Face-Recognition-App

To run the project:
 1. Clone the repo 
 
 2.
 - Navigate to facerecognitionbrain from cmd
 - Run   npm install
 - Run   npm start
 
 3.
 - Navigate to facerecognitionbrain-api from cmd
 - Run   npm install
 - Run   npm start
 - You must add your own API key in the controllers/image.js file to connect to Clarifai API.
